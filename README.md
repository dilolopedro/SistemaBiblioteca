# Sistema de Biblioteca em Java

Projeto desenvolvido em Java para demonstrar conceitos de:

- Programação Orientada a Objetos (POO)
- Encapsulamento
- Construtores
- Getters e Setters
- Listas e Coleções
- Tratamento de Exceções
- Organização em Pacotes

## Funcionalidades

- Cadastro de livros
- Cadastro de alunos
- Empréstimo de livros
- Devolução de livros
- Verificação de disponibilidade
- Limite de empréstimos
- Busca de livros
